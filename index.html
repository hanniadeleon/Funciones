Etiquetas

Etiquetas pre

La etiqueta pre sirve para respetar espacios y lineas en blanco, al contrario de la etiqueta parrafo

Hola soy hannia tengo 17 

Hola          soy
        hannia     tengo      17

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas

Según datos de la OMS el consumo excesivo de azucar provoca daños severos en la salud de los niños

Etiqueta bdo

Si se desea cambiar la dirrecion de en texto, se utiliza la etiqueta bdo

este es un texto normal

Este es un texto diferente

Etiqueta cite

La etiqueta cite se utiliza para escribir citas textuales

«Esta noche es la noche. Y va a volver a pasar, una y otra vez… tiene que pasar.» por: «Dexter»

«Es agradable tener un viejo amigo para cenar.»por: «Hannibal»
